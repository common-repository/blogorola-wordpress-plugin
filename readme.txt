goodies for [blogorola](http://www.blogorola.com "blogorola") - ultimate blogroll.* **comment synchronization**all comments are synchronized between blogorola and your blog. so any comments posted on your blog will be displayed on blogorola and vice-versa. this enables much easier discussions between readers (members of the blogorola community).comments received from blogorola will be marked with a small icon before author's name.* **hot'or'not**this will display a hot'or'not voting toolbar in your post detail. hot'or'not is blogorola's rating system, which readers can use to rate your posts. posts with good ratings are read more often and are featured on blogorola's website.* **update services**	our update service enables your wordpress blog to automatically notify blogorola of a new post as soon as you publish it. this way it will appear on blogorola almost immediately instead of waiting for the spider to crawl your blog for new posts.

* **statistics**

blog and posts status from blogorola - rate, clicks, etc.== Installation ==* unzip/upload to /plugins directory (directory blogorola)* activate via wordpress plugins tab* new tab (blogorola) appear in options page - go there* insert your unique api key - create it on blogorola.com (see FAQ)== Frequently Asked Questions ==

= where can i get api key? =* create a user profile in [registration](http://www.blogorola.com/si/registracija "registration") page
* claim your blog, follow this [guide](http://www.blogorola.com/si/prevzem-mojega-bloga "guide")
* while logged in, check your blog's detail page ([example](http://www.blogorola.com/si/blog/1904 "example"))
* your unique API Key will appear below== Screenshots ==
